Fashion Hub

Sobre o Projeto

O Fashion Hub é uma plataforma online desenvolvida para oferecer uma experiência completa de compras de roupas e acessórios. Este projeto tem como objetivo integrar funcionalidades modernas de um sistema e-commerce, focando em simplicidade, segurança e escalabilidade.

Funcionalidades Planejadas

  • Cadastro e autenticação de usuários com JWT e OAuth.
  • Gestão de produtos, incluindo criação, edição, exclusão e visualização.
  • Busca e filtragem de produtos por categorias, tamanhos e preços.
  • Sistema de carrinho de compras com cálculo de subtotal e taxas.
  • Integração com meios de pagamento para finalizar compras.
  • Painel administrativo para gestão de usuários, pedidos e estoque.
  • API documentada com Swagger para integração com outras aplicações.
  • Testes automatizados utilizando Jest para garantir qualidade do código.
